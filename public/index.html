Toys - API Documentation



Toys API

HTTP Method Route Description Parameters/Request Body
Get /toys Returns a list of toys according to the query parameters
  • page (optional): page number to return (default: 1)
    https://gal-list-toys.cyclic.app/toys?page=1
  • limit (optional): number of toys per page (default: 5, maximum: 10)
    https://gal-list-toys.cyclic.app/toys?limit=2
  • sort (optional): field to sort by (default: price)
    https://gal-list-toys.cyclic.app/toys?sort=price
  • reverse (optional): if set to yes, sorts in descending order
    https://gal-list-toys.cyclic.app/toys?sort=price&reverse=yes
  • s (optional): the search query to match against toy names, infos or categories
    https://gal-list-toys.cyclic.app/toys?s=Building
Get /games/prices Returns a list of toys filtered by price range
  • page (optional): page number to return (default: 1)
    https://jittery-seal-leg-warmers.cyclic.app/toys/prices?page=1
  • limit (optional): number of toys per page (default: 5, maximum: 10)
    https://gal-list-toys.cyclic.app/toys/prices?limit=2
  • min (optional): minimum price to filter by (default: 0)
  • max (optional): maximum price to filter by (default: Infinity)
    https://gal-list-toys.cyclic.app/toys/prices?min=10&max=30
GET /toys/single/:id Returns a single toy by its ID
  • id: the ID of the toy to retrieve
    https://gal-list-toys.cyclic.app/toys/single/64f0420d5b5c900cea9c4643
POST /toys Adds a new toy,
Token is required
  • Request Body: Toy data to be added in the following format:
    {
    name: String, // 2-100 characters
    info: String, // 2-400 characters
    category: String, // 2-400 characters
    ing_url: String, // 2-400 characters, allowed but not required
    price: Number // 1-99999
    }
PUT /toys/:id Updates an existing toy,
Token is required
  • id: the ID of the toy to update
  • Request Body: Toy data to be added in the following format:
    {
    name: String, // 2-100 characters
    info: String, // 2-400 characters
    category: String, // 2-400 characters
    ing_url: String, // 2-400 characters, allowed but not required
    price: Number // 1-99999
    }
DELETE /toys/:id Deletes a toy by its ID,
Token is required
  • id: the ID of the toy to delete
